```
Test11_4.java
```

## 정수의 표현 

```
1) 10진수 정수

      System.out.println(100);
      => 100

2) 8진수 정수 : 0으로 시작

      System.out.println(0144);
      => 100

3) 16진수 정수 : 0x 또는 0X로 시작

      System.out.println(0X64);
      System.out.println(0x64);
      => 100

4) 2진수 정수 : 0b 또는 0B로 시작. 1 앞의 0 생략 가능

      System.out.println(0b01100100);
      System.out.println(0B01100100);
      System.out.println(0b1100100);
      => 100

```
