# 자바 소스 파일 기본 구조

## 관련 파일

```
Test10_1.java
```

## 1) 클래스 소속 패키지 선언 

```
- 선언 형식 : package 패키지명;
  ex. package bitcamp.java100;
```

## 2) 클래스 선언

```
- 선언 형식 : public class 클래스명{} 
  ex. public class Test10_1{}
```

## 클래스 실행 블록 선언

```
- 선언 형식 : 
  public static void main(String[] args){
      System.out.pringln("Hello!");
  }
```
